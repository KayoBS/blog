Relembrando o post anterior...

Para começar esse novo post, cai bem relembrar brevemente o que foi feito na versão anterior desse projeto, que pode ser encontrada aqui. Basicamente, a calculadora se resumia a quatro funções para as operações [sum(), sub(), mtl() e dvs()] e uma leitura dos argumentos passados para a função main(), que seguiam a seguinte forma, ignorando a chamada do programa no terminal: primeiro operando + operador + segundo operando.

Exemplo: ./main.out 3 . 2

A saída no terminal será 6, pois "." é o operador de multiplicação que, como já explicado, não é o convencional asterisco por estranhos problemas que ele gerava na execução. Em um futuro próximo ainda verei o porquê desses problemas, para trocar essa especificidade pelo que é convencional e comum de se usar, mas era assim e por enquanto continuará sendo assim.

Era apenas isso que a antiga versão do projeto fazia - lia dois operandos, um operador e imprimia o resultado na tela. Ainda faltava o principal que era a leitura de arquivos e muitas coisas básicas, como a separação dos códigos em diferentes arquivos. Mas isso era até pouco tempo atrás...

Primeiras alterações

A primeira coisa que eu fui fazer, que era uma das coisas mais importantes de se fazer (se não, a coisa mais importante) e que eu não tinha feito até então era a separação das funções em diferentes arquivos. Isso é especialmente necessário para que o código fique mais organizado, em espécies de "pacotes de código", o que ajuda em muito a fazer do projeto mais expansível.

Não é nada de outro mundo, apenas fiz dois arquivos adicionais, que no caso eram o operations.hpp e o operations.cpp, respectivamente declaração e definição, dentro de uma pasta operations/ dentro de outra chamada de lib/. Se eu tivesse apenas criado um arquivo .hpp que incluísse as declarações e as definições, não haveria a necessidade de mudar o comando de complilação, mas como o fiz, ficou da seguinte forma: g++ main.cpp lib/operations/operations.cpp -o main.out

                    //operations.hpp
#ifndef FSTCALC_OPERATIONS_HPP_INCLUDED
#define FSTCALC_OPERATIONS_HPP_INCLUDED

int sum(int, int);
int sub(int, int);
int mlt(int, int);
double dvs(int, int);

#endif
                
                    //operations.cpp
#include "operations.hpp"

int sum(int a, int b) { return (a + b); }
int sub(int a, int b) { return (a - b); }
int mlt(int a, int b) { return (a * b); }
double dvs(int a, int b) { return (static_cast(a) / b); }

Além disso, tendo em vista que eu teria que criar outra biblioteca autoral mais para a frente (não lembro se fiz isso antes ou depois de escrever essa dita outra biblioteca), criei o arquivo lib.hpp dentro daquela pasta lib/ para incluir todos os arquivos que eu fosse criar com apenas uma linha de código no arquivo principal.

Assim, pude simplesmente escrever #include "lib/lib.hpp" no meu main.cpp
